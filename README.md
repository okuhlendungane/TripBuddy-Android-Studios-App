
## Trip Buddy App – Android Project (Java)

### Overview

Trip Buddy is an Android mobile application I developed as part of my **second-year Software Engineering studies**. The app is designed to help users plan and organize trips by capturing key details, making selections, and displaying a clear summary of the trip.

This project was built using **Java in Android Studio**, with a focus on applying core mobile development concepts such as activity navigation, user input handling, and UI design.

---

### Key Features

* Multi-screen app structure (Splash Screen, Main Activity, Summary Activity)
* User input handling (text fields, selections, and basic validation)
* Navigation between activities using intents
* Spinner component for selecting predefined options
* Summary screen displaying compiled trip information
* Basic UI styling using XML layouts

---

### What I Learned

* How Android apps are structured using activities and the lifecycle
* Passing data between screens using **Intents**
* Designing user interfaces with XML layouts
* Handling user input and displaying dynamic data
* Structuring code for readability and maintainability
* Debugging errors in Android Studio

---

### Challenges Faced

* Understanding how to properly navigate between multiple activities
* Managing data flow between screens (especially passing user inputs correctly)
* Working with the **Spinner component**, which was initially confusing in terms of setup and handling selections
* Debugging layout issues and unexpected crashes
* Getting comfortable with Android Studio’s environment and project structure

---

### How I Overcame Challenges

* Broke down each activity and focused on one screen at a time
* Practiced using intents repeatedly until the concept became clear
* Used trial and error to understand how UI components behave
* Simplified complex parts (like spinner logic) before improving them
* Referred to documentation and examples to reinforce understanding

---

### Reflection

As a **second-year project**, Trip Buddy played an important role in strengthening my foundation in Android development. It helped me transition from basic programming concepts to building a complete, multi-screen mobile application. The challenges I faced improved my problem-solving skills and gave me more confidence working in a real development environment.
